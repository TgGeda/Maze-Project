The 3D Maze Project

A 3D computer graphics game using raycasting; a technique for rendering 3D objects.

Features

3D Graphics

The maze is rendered in 3D. This feature creates a realistic and immersive experience.

Map

This feature shows the player where they are in the maze, and it can also show the player the different paths that they can take.

Weapons & Enemies

This feature makes the game more adventurous

About

Project Inspiration

The primary source of inspiration for this project stems from my deep affection for vintage 3D computer games, specifically virtual cops and GTA Vice City. Drawing upon the immersive nature of movies and real-life encounters, which enhanced the allure and authenticity of 3D games, I recognized this as a chance to gain insights into the intricacies of their creation and acquaint myself with the underlying technologies.
